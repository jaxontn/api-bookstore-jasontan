#########TO RUN FROM ANOTHER LAPTOP###########
1. Install Dependencies
    npm install  

2. Run the Database:
    docker-compose up --build

3. Create a Migration: 
    npx prisma migrate dev --name init

4. Generate Prisma Client: 
    npx prisma generate


5. Run the NestJS Web App
    npm run start:dev

6. View database data
    npx prisma studio
